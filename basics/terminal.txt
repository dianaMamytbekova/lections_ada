ls - показывает все файлы/директорий(папки)в текущей директорий
ls test - показываетт все файлы и папки внутри указанной директорий "test"
ls -a - показывает все файлы и директорий, а также все скрытые файлы/директорий
ls -l - показывает более подробную информацию о файлах и папках
ls -la - показывает более подробную информацию еще о скрытых файлах/папках

cd - переходит в ломашнюю папку/директорию
cd "test" - переходит в заданную территорию (в нашем случае тест)
1) зайти обратно в рабочую дирекеторию, из домашней лиректорий
cd .. - переходит на одну директорию назад
cd ../.. итд - переходит на 2 или более директорий назад (чтобы перейти еще, можно через / указывать по 2 точке)
mkdir test - создает директорию test (именно в текущей директорий)
mkdir test1/test2 - создает папку test2  в папке test1 (если директорий тест 1 не существует то выйдет ошибка)
mkdir test1 test2 test3 test4 - создает 4 директорий (test1-test4)в текущей папке

touch text.txt - команда которая создает фаил с любым расширением в текущей директорий 
touch test1.py test1.txt test2.py tets2.txt - создает много файлов с разными расширениями одновременнно

rm test.txt - удаляет фаил test.txt, если такого файла нет - выкинет ошибку, если указать директорию - так-же выкинет ошибку
rmdir - удаляет директорию, удаляет только пустую директорию
rm -rf - удаляет файл или папку со всем содерержимым удаляет (безовзвратно)

mv test.txt test1/ - перемещает файл test/txt в директорию test
mv test.txt test2.py - перемещает файл tesst.txt в test2.py
mv test/ lesson/

history - показывает все команды которые были выполнены в этом терминале
history 10 - показывает последние 10 команд
history|grep cd - выводит все команды, в которых есть cd

pwd - показывает полный путь (начиная с корневой папки /)до текущей директорий

nano test.txt - открывает файл test/txt в редакторе кода nano (если такого файла нет, то создает его)
чтобы выйти из nano:
       1) cntrl + x
       2) y
       3) ENTER

cat test.txt - вводит в терминал все содержимое файла 
code . - открывает редактор кода vs в текущей директории

sudo - команда которая дает права суперюзера (super user do)
